R+ = the set of nonnegative real numbers
N = { 0, 1, 2, 3, … }

Let f, g, h: N → R+.
  • f(n) = O(g(n)) if ∃c, n0 ∈ N s.t. ∀n ≥ n0, f(n) ≤ c g(n). The growth rate of f(n) is less than or equal to that of g(n).
  • f(n) = o(g(n)) if limn→∞[f(n)/g(n)] = 0. The growth rate of f(n) is less than that of g(n).
  • f(n) = Θ(g(n)) if ∃c, c', n0 ∈ N s.t. ∀n ≥ n0, c g(n) ≤ f(n) ≤ c' g(n). The growth rate of f(n) is equal to that of g(n).
  • f(n) = g( O(h(n)) ) if ∃ h': N → R+ s.t. f(n) = g(h'(n)) and h'(n) = O(h(n)).
